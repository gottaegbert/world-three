## Run Locally

Clone the project

```bash
  git clone https://github.com/ricardoiso/world-three.git
```

Go to the project directory

```bash
  cd world-three
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```


## Tech Stack

**Front:** ThreeJS, Vite, TailwindCSS

**Server:** Node
